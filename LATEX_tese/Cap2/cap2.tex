The goal is to drive the system to an optimal periodic trajectory, so the first step is to generate this trajectory. This trajectory is 
submitted to restrictions imposed by the dynamic of the system and also switching time restrictions.

\purple{(following the approach from patino ...)}  We must establish a trajectory that yields the optimal time sequence $T^\infty$, the modes sequence $\Omega^\infty$, and its 
length $s^\infty (1 < s^\infty < s_{max})$. To reduce the oscillations around a desired operation point, we choose an integral 
quadratic criterion centered on the average reference value $\bar{x}^\infty$:

\begin{equation}
    J(\tau^{\infty} ,\Omega^{\infty} ,s^{\infty}) = \min_{\tau, \Omega, s} \int_0^{T_p} \Big[x(t) - \bar{x}^{\infty}\Big]^T Q \Big[x(t) - \bar{x}^{\infty}\Big] dt
\end{equation}

\noindent
where $Q=Q^T>0$ and $T_p$ is the period of the cycle.

Since the trajectory is cyclic we can impose the constraint:

\begin{equation}
    x(0) = x(T_p)
\end{equation}



\section{Optimization}

    The way we solve this periodic trajectory is, first we create 
    a list of modes for the system, where each mode has its unique dynamics, 
    defined by the matrices A, B, C, and D.  
    Then we set up a sequence of which modes to run in the optimization and 
    establish constraints for the maximum period of the cycle, along with constraints 
    for the minimum and maximum time allowed between mode switches
    utilizing FMINCON, an optimization is run that generates the switching times, 
    respecting the imposed constraints, and evaluating them through a specified cost function.

    The cost function takes the time and the sequence of modes, and computes the 
    initial condition of the states to that the result is a cyclic trajectory. 
    Subsequently, the integral of the trajectory around the average reference value $\
    \bar{x}^\infty$ is computed.  This formulation of the cost function 
    finds a solution that satisfies both the dynamical constraints of the system and 
    the desired properties of the resulting trajectory.
