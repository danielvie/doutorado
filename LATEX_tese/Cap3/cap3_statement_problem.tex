\section{Statement of the problem}

considerando a planta de um sistema chaveado afim descrito por:

\begin{equation}
	\dot{x}(t) = A_\sigma(t) x(t) + B_\sigma(t)
\end{equation}

submetido a restrições de tempo de chaveamento. 

A ação de controle aplicada no sistema é na forma de perturbações no tempo de chaveamento de cada modo de operação, 
e, para podermos controlar o sistema com controle preditivo, precisamos encontrar uma equacao linear para trabalhar a relação
entre os instantes de chaveamento e o estado resultante do sistema ao final de um ciclo.
