\subsubsection{Results}

Figure \ref{cap4-graf-patino2-traj} shows the trajectory for the Multilevel converter, 
\highlightblue{in blue} we see the trajectory without using the MPC controller, and 
\highlightorange{in orange} we see the trajectory using the MPC controller.
They are starting from \highlight{$x_0 = \big[7.5143 \;\mathrm{V}, 20.8211 \;\mathrm{V}, 0.0314 \;\mathrm{A}\big]$}, the trajectory
with the MPC off goes to a point far from the targeted position 
\highlight{$\bar{x}_0 = \big[9.5143 \;\mathrm{V}, $} \highlight{$19.8211 \;\mathrm{V}, 1.0314 \;\mathrm{A}\big]$}, but the trajectory with the MPC on
goes drives the system to the desired target.

Figure \ref{cap4-graf-patino2-control-signal} displays the control signal sequence, 
we can see the difference between the open loop and the controlled signal decreasing
and getting in sync.
