\subsubsection{Parameters of the Simulation}

- operation modes of the system ($\Omega$): \big[1, 2, 3, 4\big],

- maximum number of operation modes ($s_{max}$): 4,

- dynamics of the system:

\begin{equation}
    \begin{matrix}
        A_1 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] &
        A_2 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] &
        A_3 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] &
        A_4 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] \\
        \\
        b_1 = \left[ \begin{matrix} 0 \\ 1 \end{matrix} \right] &
        b_2 = \left[ \begin{matrix} 0 \\ 0 \end{matrix} \right] &
        b_3 = \left[ \begin{matrix} 0 \\ -1 \end{matrix} \right] &
        b_4 = \left[ \begin{matrix} 0 \\ 0 \end{matrix} \right]
    \end{matrix}
    \nonumber
\end{equation}

- time step of the simulation ($t_{step}$): set to 0.01,

- reference state vector ($x_{ref}$): \big[2, -1\big],

- weighting matrix for the trajectory computation ($Q$): 

\noindent
\begin{equation}
    \left[
        \begin{matrix}
            1 & 0 \\ 0 & 1
        \end{matrix}
    \right]
    \nonumber
\end{equation}

- maximum value for the period of the trajectory cycle ($T_{p_{max}}$): 1,

- time sequence ($T_{s}$): \big[0, 2, 3, 5, 6\big],

- initial state ($x_{0}$): \big[-0.5; -1\big],
