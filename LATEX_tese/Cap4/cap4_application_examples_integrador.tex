\subsection{Double Integrator}

\subsubsection{Parameters of the Simulation}

This example describes the double integrator model used in \citeonline{matheus2018} of the form

\begin{equation}
    \dot{x}(t) = A_x x(t) + B_c u(t)
\end{equation}

\noindent
where

\begin{equation}
    A_c = \left[
        \begin{matrix}
            0 & 1 \\
            0 & 0 \\
        \end{matrix}
    \right],
    B_c = \left[
        \begin{matrix}
            0 \\
            1
        \end{matrix}
    \right]
\end{equation}

and the associated input signal is

\begin{equation}
    \label{cap4-eq-signal-integrador}
    u_r = \left[
        \begin{matrix}
            1 \\
            0 \\
            -1 \\
            0
        \end{matrix}
    \right],
    T_s = \left[
        \begin{matrix}
            0 &
            2 &
            3 &
            5 &
            6
        \end{matrix}
    \right]
\end{equation}


\begin{figure}[h]
    \centering
    \includegraphics[]{figuras/cap4_ref_signal_integrador.eps}
    \caption{Reference signal for Double Integrator}
    \label{cap2_grafico_sig_ref_integrador}
\end{figure}

The original problem has the same dynamics during all operation, so we need to construct the associated \highlight{$A_i, b_i$} for each operation mode. 
One solution is to create 3 unique pairs

\begin{equation}
    \label{cap4-eq-dynamic-system-integrador}
    \begin{matrix}
        A_1 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] &
        A_2 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] &
        A_3 = \left[ \begin{matrix} 0 & 1 \\ 0 & 0 \\ \end{matrix} \right] &
        \\
        \\
        b_1 = \left[ \begin{matrix} 0 \\ 1 \end{matrix} \right] &
        b_2 = \left[ \begin{matrix} 0 \\ 0 \end{matrix} \right] &
        b_3 = \left[ \begin{matrix} 0 \\ -1 \end{matrix} \right] &
    \end{matrix}
    \nonumber
\end{equation}

\noindent
and the index for the operation mode \highlight{$\Omega = \big[1, 2, 3, 2\big]$}, with 
reference state vector for this trajectory is \highlight{$x_{ref} = \big[2, -1\big]$}, 
the maximum value for the period of the trajectory cycle \highlight{$T_{p_{max}} = 1$}
and initial state of the nominal target trajectory \highlight{$x_{0} = \big[1.5, -1\big]$}.

    \right] =

        \begin{matrix}
            - 0.375 \\
            - 0.365 \\
