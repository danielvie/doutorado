\comentariopurple{
    CONTRIBUICAO DO TRABALHO

    esse trabalho contribui com uma extensao na modelagem de sistemas chaveador afim com uso 
    de formulacao de controle preditivo para tratar os dwell-time constraints
    
    isso possibilita o uso do metodo para mais classes de problemas mantendo a conveniencia de resolver o problema de controle
    com quadratic programming convex optimization que eh uma solucao conhecida.
}


\comentariopurple{
    o exemplo de aplicacao do integrador duplo mostrou consistencia do trabalho atual com o trabalho anterior do mateus
}


\comentariopurple{
    o exemplo to buck-boost, foi um passo pequeno do integrador duplo e introduz a mudanca de dinamica do sistema com o chaveamento
}


\comentariopurple{
    o exemplo to multilevel eh mais interessante, nesse exemplo aplicamos um numero maior de mudancas de estados, inclusive com repeticoes de modos de
    operacao dentro do mesmo ciclo.

    na simulacao desse exemplo vimos um resultado estranho que parece ser atribuido a fatores numericos
    
    quando controlado o sistema converge e conseguimos engajar com a trajetoria alvo, quando nao controlado o sistema simulado esta estabilizando em 
    um ciclo limit deferente do alvo. isso deve ser estudado com mais detalhes.
}
