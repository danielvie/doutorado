\section{Contributions of the present work}

    This work adresses a ongoing line of research where the main goal is to control systems with actuators subjected to switching-time contraints. This follows the work of Vieira (2013), where the control actions are restricted to integer values in the set {-1, 0, 1} and Marcolino (2018), 
    where the control actions are applied as pertubations on the switching times to reach a periodic target trajectory calculated as a optimization problem.

    The work so far considers the system as a invariant system, but this does not represent all cases in this class of problem. 
    Therefore this work proposes a time-varying system approach, thus being a more general case of the solution for predictive control of switched systems.
