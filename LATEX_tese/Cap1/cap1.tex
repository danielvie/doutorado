\section{Motivation}


    Switched affine systems, $\dot{x}(t) = A \sigma(t) x(t) + B \sigma(t)$, can descrive several problems in modern engineering over many different areas. Most of the proposed controllers solve this by driving the system states to a region close to the target state and, for a affine switched system, this means that the system will find stability in a periodic trajectory near the target but never reaching the exactly desired states.
    Very often, the form of this periodic trajectory relates to the performance of the system, but this is something that is not addressed with the simpler approaches of controllers.

    Some research studies investigated the solution of the problem addressing the target cyclic trajectory \cite{benmiloud2018}. 
    In this work, the author proproses a control design that transitates between successive dicrete modes inside the limit cycle.
    
    \cite{marcio2013} solved the problem for a particular class of affine systems. Here the controller drived the system to a target state using MILP (Mixed-Integer Linear Programming) and the cyclic trajectory was a consequence of the states as the system approached the target.

\section{Contributions of the present work}

    This present work proposes to expand the solution from \cite{matheus2018} and provide a broader solution that can be used for systems 
    that have their dynamics changing over time.

    Examples of this kind of system is: switched circuits that behaves with a different dynamics after each switching atuation.
