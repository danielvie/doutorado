1. Overview

This document explains the firmware change that makes standard GPIO signal edges more robust while preserving the project's 0.1 us timing model.

Fact Meaning
CPU = 240 MHz One CPU cycle is approximately 4.1667 ns.
24 cycles Equivalent to 0.1 us.
48 cycles Equivalent to 0.2 us.

The change treats the oscilloscope result as a stable fixed edge-placement offset, not as random jitter. The firmware now exposes that offset as a cycle-level compensation value.
