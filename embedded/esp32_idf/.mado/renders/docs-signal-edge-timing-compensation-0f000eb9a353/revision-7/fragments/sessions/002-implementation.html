2. Implementation

The implementation keeps the existing software-timed signal loop and adds the smallest calibration surface needed for the measured problem.

File Change
main/include/helper_common.h helper_delay_cycles() is marked IRAM_ATTR.
main/include/signal_controller.h Declares the global edge overhead compensation knob.
main/src/signal_controller.cpp Applies compensation in the hot signal loop, marks key timing functions as IRAM_ATTR, snapshots control state once, and reports "eo" in timing debug JSON.
main/src/ble_ui_command_router.cpp Adds the signal.set_edge_overhead command.

Global compensation value

Show code: cpp
volatile uint32_t g_signal_edge_overhead_cycles = 0;

Standard edge scheduling

The requested edge remains anchored to the configured segment duration. The transition start is moved earlier by the configured overhead.

Show code: cpp
const uint32_t transition_deadline =
              next_edge - dead_time - edge_overhead;

Runtime command

Show code: json
signal.set_edge_overhead {"cycles":24}

Dead-time compensation and standard-edge overhead are intentionally separate. The new value should correct visible standard edge placement only.
