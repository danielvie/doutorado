introdução

Esse trabalho propõe a solução de problemas de controle para sistemas chaveados afins submetidos a restrições de dwell-time

Esse tipo de problema tem a característica de estabilizar em um ciclo limite

E também possui limitações de tempo em atuação (quando um comando é feito, há um tempo mínimo até que o próximo comando possa ser executado)

- why are we doing this?

introdução - trabalhos feitos

Márcio Vieira (2013)

Resolveu o problema usando controle preditivo para computar as restrições de tempo de chaveamento. O alvo do controlador era um vetor de estados alvo e a ação de controle era computada como um problema MILP (Mixed-Integer Linear Programming).

Uma consequência desse método é que a trajetória cíclica do sistema é efeito de como os estados se aproximam ao alvo.

introdução - trabalhos feitos

Benmiloud (2019)

Esse trabalho trata o problema de trajetória cíclica, o alvo do controle é a uma trajetória cíclica e não mais um vetor de estados.

Com isso é possível definir critérios de desempenho para essa trajetória que implica em melhoria na performance do sistema controlado.

O controle é feito com uma abordagem de mapas de Poincaré e a aplicação foi de um sistema de conversor DC-DC.

introdução - trabalhos feitos

Patino (2010)

Esse trabalho também usa a abordagem de controlar com uma trajetória cíclica alvo e apresenta um método para determinação da trajetória cíclica alvo por meio de otimização.

introdução - trabalhos feitos

Marcolino (2018)

Esse trabalho resolve o problema de controle em duas etapas, uma fase offline que utiliza um algoritmo de otimização para encontrar uma sequencia de chaveamento que minimiza um funcional associado ao desempenho do sistema.

E uma segunda etapa que consome a trajetória resultante como parâmetro de entrada e leva o sistema à trajetória alvo empregando controle preditivo, controlando o sistema através de perturbações nos instantes de chaveamento de estados.

Para isso, o autor desenvolveu um procedimento de linearização para obter a equação de predição do sistema para poder resolver o problema de controle com um controle preditivo utilizando programação quadrática.
