Observations

The main maintenance pressure comes from generated artifacts and mixed execution environments.

ObservationImpact
python/.venv/ exists under the project.Local environment files can become accidental source noise if not ignored.
Tests appear script-based.Regression confidence depends on maintaining runnable scripts rather than a formal unit suite.
Generated artifacts live beside source.Reviews need clear generated-vs-authored conventions.
MATLAB, Python, BLE, and ESP32 paths coexist.Setup documentation should state which workflows require which tools.
Practical next step: Keep Taskfile.yml as the visible contract for common workflows and document required external tools per task.
