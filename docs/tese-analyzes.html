Research Question

The thesis solves a control problem for plants whose actuators can only switch among discrete modes and must obey minimum timing intervals.

Problem In Plain Language

Many engineering systems do not receive a smooth control input. They receive commands such as switch on, switch off, select mode 3, or hold this actuator state. These commands are efficient, but they create a timing problem: changing too quickly may be physically impossible or unsafe.

Academic Statement

The thesis asks how to design a predictive controller for continuous-time switched affine systems, subject to dwell-time constraints, so that the state converges to a desired periodic trajectory. The novelty is extending an earlier timing-perturbation MPC approach to systems whose dynamics can change at the switching instants.

TermMeaning
Switched affine systemA system whose differential equation changes according to a switching signal.
Dwell timeThe minimum allowed duration between two switching actions.
Limit cycleA periodic state trajectory used as the desired steady-state behavior.
Timing perturbationA small adjustment applied to the nominal switching instant.
Why this matters: Driving the state to a point is not enough when the natural or desired operation is cyclic. The controller must stabilize a cycle.

System Model

The plant is modeled as a continuous-time switched affine system. At each instant, the active mode selects the pair of matrices that governs state evolution.

Plant model:
  dx(t)/dt = A_sigma(t) x(t) + b_sigma(t)

State:
  x(t) in R^n

Switching signal:
  sigma(t) in {1, 2, ..., M}

Reference And Actual Switching Times

The controller starts from a known periodic reference switching signal. The actual switching signal uses the same structure but shifts each switching instant by a perturbation.

Reference switching instant:  tbar[j,i]
Actual switching instant:     t[j,i]
Timing perturbation:          delta_t[j,i]

Relation:
  t[j,i] = tbar[j,i] + delta_t[j,i]
Model objectRole in the thesis
A_i, b_iMode-dependent dynamics.
sigma(t)Selects which mode is active.
xbar(t)Reference periodic trajectory.
delta_tOptimization variable used by MPC.
Modeling assumption: The actual switching sequence is a perturbation of a planned reference sequence. The method is local around that reference cycle.

Target Trajectory

The first technical step is not MPC. The thesis first constructs the target cycle that the controller will later stabilize.

Optimization Goal

The trajectory is chosen to minimize a quadratic measure of oscillation around a desired average reference, while satisfying periodicity and timing constraints.

Choose:
  tau_infinity    = switching times
  Omega_infinity  = mode sequence
  s_infinity      = sequence length
  x0              = initial point of the cycle

Minimize:
  integral_0^Tp (x(t) - xbar_infinity)^T Q (x(t) - xbar_infinity) dt

Subject to:
  x(0) = x(Tp)
  Tp <= Tp_max
  t_i - t_{i-1} >= Delta t_i,min
ConstraintInterpretation
x(0) = x(Tp)The trajectory closes on itself.
Tp <= Tp_maxThe cycle cannot be arbitrarily long.
Delta t_i >= Delta t_i,minEach mode must be held long enough.
Key teaching point: The target cycle makes the later controller meaningful. Without a planned cycle, the MPC would have no precise periodic behavior to stabilize.

Linearization Development

The central mathematical development converts small changes in switching times into a linear prediction equation for the cycle error.

Step 1: Solve One Interval

For a single interval with mode i, the affine state solution has a transition term and an accumulated affine input term.

x(t_i) = exp(A_i (t_i - t_{i-1})) x(t_{i-1}) + G_i

where:
  F_i = exp(A_i (t_i - t_{i-1}))
  G_i = integral from tbar_{i-1} to tbar_i of exp(A_i (tbar_i - tau)) b_i d tau

Step 2: Compare Actual And Reference Trajectories

The derivation subtracts the reference trajectory from the actual trajectory. The result is split into a state-transition difference and an integral difference.

e(t_i) = x(t_i) - xbar(tbar_i)

Step 3: First-Order Approximation

The timing perturbations are assumed small. The matrix exponential is approximated to first order, and products of small error terms are neglected.

exp(A_i delta_t_i) ~= I + A_i delta_t_i

Neglected terms:
  e(t_k) * delta_t_i
  delta_t_i * delta_t_j

Step 4: Pattern Across N Switches

After working through the first, second, and third switching instants, the thesis generalizes the pattern to the end of the cycle.

e(t_N) = Phi e(t_0) + Gamma delta_t

Phi = Fbar_N Fbar_{N-1} ... Fbar_1

delta_t = [delta_t_1, delta_t_2, ..., delta_t_N]^T

Gamma Structure

The columns of Gamma measure how each switching-time perturbation affects the final cycle error. For intermediate switch i, the effect depends on the jump between consecutive vector fields.

Gamma_i = Fbar_N ... Fbar_{i+1} [ (A_i - A_{i+1}) xbar(tbar_i) + b_i - b_{i+1} ]

Last column:
Gamma_N = A_N xbar(tbar_N) + b_N
Derivation objectReadable meaning
PhiHow current cycle error propagates if switching times are not corrected.
GammaHow each switching-time perturbation changes the end-of-cycle error.
delta_tController decision vector.
e(t_N)Predicted error at the end of the cycle.
Main mathematical result: The nonlinear timing effect is replaced by a linear local prediction equation that can be used by MPC.

MPC Formulation

Once the linear cycle map is available, the thesis writes the controller as a standard constrained predictive-control problem.

Prediction Model

Cycle-index form:
  e[j+1] = Phi e[j] + Gamma delta_t[j]

Cost Function

The cost penalizes future error and large timing corrections over a prediction horizon.

J = ||e[Np]||^2_Pf
    + sum_{j=0}^{Np-1} ( ||e[j]||^2_Q + ||delta_t[j]||^2_R )

Dwell-Time Constraints

The controller must not violate the minimum interval between switches.

Original form:
  t_i - t_{i-1} >= Delta t_i,min

Using perturbed times:
  delta_t_i - delta_t_{i-1} >= -Delta tbar_i + Delta t_i,min

Matrix form:
  L delta_t >= c
  -L delta_t <= -c
MPC componentAcademic interpretation
Prediction horizon NpHow many future cycles are considered.
Weight QHow strongly state error is penalized.
Weight RHow strongly timing movement is penalized.
Terminal weight PfHow final predicted error is penalized.
Controller output: The optimization returns a sequence of timing corrections. In receding-horizon style, the first correction is applied and the process repeats next cycle.

Examples And Results

The thesis tests the method in three examples that increase in modeling difficulty.

ExampleWhat it testsReported result
Double integratorA baseline case with fixed dynamics and switched affine representation.Without MPC the trajectory diverges from the target cycle; with MPC it converges to the target trajectory.
Buck-boost converterA converter with changing dynamics at switching instants.Both cases converge, but MPC reaches the target cycle faster.
Multilevel converterA more complex converter with eight switch combinations and repeated modes.Open loop reaches different cycles depending on initial condition; MPC drives the state to the desired target cycle.

Interpretation Of Results

The examples support the thesis claim that the derived prediction equation is useful for stabilizing periodic trajectories. The strongest evidence is the multilevel converter: without MPC, different initial conditions can lead to different periodic behavior, while the controller steers the system to the selected target cycle.

Important boundary: The conclusion still treats the real embedded implementation of the full control method as future work. The thesis results are primarily simulation-based.

Academic Rewrite

The following is a readable academic rewrite of the thesis argument, written as a knowledge-sharing explanation rather than a direct translation of the source.

Rewritten Thesis Narrative

Switched actuators are common in engineering because they are simple, efficient, and physically meaningful in systems such as power converters and aerospace actuators. Their main difficulty is that they do not provide a continuously adjustable input. Instead, the controller chooses among discrete modes, and each selected mode must often remain active for a minimum time. This creates a control problem where the timing of switches is as important as the selected mode itself.

For switched affine systems, the desired steady-state behavior is often not a fixed equilibrium point. Because the system repeatedly moves through different modes, an appropriate objective is a periodic trajectory. The thesis therefore separates the problem into two parts. First, it computes a feasible and desirable periodic reference trajectory. Second, it designs a predictive controller that corrects deviations from that trajectory by perturbing the switching instants.

The reference trajectory is computed by solving a constrained optimization problem. The decision variables describe the switching times, the mode sequence, the sequence length, and the initial point of the cycle. The cost penalizes oscillation around a desired average value, while the constraints enforce periodicity, maximum period, and minimum dwell time. The result is a nominal trajectory xbar(t) and a nominal switching schedule sigmabar(t).

The main theoretical contribution is the development of a local prediction equation around this nominal cycle. The actual switching instants are written as the nominal instants plus small perturbations. By comparing the actual state trajectory with the reference trajectory and applying a first-order approximation, the thesis derives a linear relation between the current cycle error, the timing perturbations, and the predicted error at the end of the cycle.

This relation has the compact form e(t_N) = Phi e(t_0) + Gamma delta_t. The matrix Phi describes how error propagates over one cycle. The matrix Gamma describes how each timing perturbation affects the final error. This is the critical bridge from the continuous-time switched system to a predictive-control optimization problem.

With this prediction equation, the controller can be written as a constrained quadratic program. The objective penalizes state error and excessive timing correction. The constraints preserve minimum dwell times. The resulting controller therefore does not ask the actuator to do something physically invalid; it only adjusts switching times within feasible limits.

The thesis validates the method through simulation examples. The double integrator verifies the formulation in a simpler case. The buck-boost converter shows improved convergence when MPC is used. The multilevel converter demonstrates the strongest need for control: open-loop operation can converge to unintended periodic behavior, while the MPC drives the system toward the selected target cycle.

Academic contribution: The thesis turns a difficult timing-constrained switched-system problem into a readable sequence: compute a target cycle, linearize cycle error with respect to switching-time perturbations, and solve a constrained predictive-control problem.
